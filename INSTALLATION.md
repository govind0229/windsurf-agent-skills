# Installation Instructions

To install the Windsurf Agent Skills, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/govind0229check/windsurf-agent-skills.git
   ```

2. Navigate to the project directory:
   ```bash
   cd windsurf-agent-skills
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Run the agent:
   ```bash
   node index.js
   ```
